# Windows.UI.Xaml.Controls.RichEditBox.ContextMenuOpening

## -description

Occurs when the system processes an interaction that displays a context menu.

## -xaml-syntax

```xaml
<RichEditBox ContextMenuOpening="eventhandler" />
```
